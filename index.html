Gantt Update

Sincroniza automáticamente los planes de proyecto en Google Sheets con el estado real de los tickets en Jira, cada 2 horas, sin intervención manual.

Última actualización 10 de junio, 2026
Owner Diego Ferrocchio
Plataforma Google Apps Script
Estado ● Activo
📅
Jira → Google Sheets Sync
Lee el ID de ticket Jira en cada fila del Gantt y actualiza estado, % de avance y fechas reales usando la API de Jira.
Activo ▾
Activo
Pendiente
Backlog
Deprecado

Schedule

every 2 hours Time-driven trigger — Google Apps Script

Herramientas

Google Apps Script Jira REST API v3 Google Sheets API JavaScript ES5

Flujo

Time trigger (2hs) → Lee col B (ticket IDs) → Jira API + Changelog → Actualiza Sheet

Columnas sincronizadas

B — Jira Ticket ID (input) D — % de avance E — Estado H — Fecha inicio real I — Fecha fin real

Configuración

JIRA_EMAIL JIRA_TOKEN JIRA_DOMAIN SPREADSHEETS[ ]

Reglas de mapeo de estado

Jira status             →  Sheet (col E)       % avance (col D)
Tareas por hacer        →  01 - Por hacer       0%
En Progreso             →  02 - En Progreso     25%
Review                  →  02 - En Progreso     50%
Control de calidad      →  03 - QA              75%
En Progreso + Bexi*     →  03 - QA              75%
Finalizado              →  04 - Completo        100%

* Regla especial: si el assignee contiene "Bexi" y el estado es "En Progreso",
  se mapea como QA al 75%.
  Estados no mapeados son ignorados — la fila no se modifica.

Lógica de fechas (changelog de Jira)

H: primera transición a "En Progreso" o "DOING" — se escribe una sola vez H: vacía si estado es "01 - Por hacer" I: última transición a "Finalizado" — se actualiza siempre I: vacía si el ticket no está en estado Finalizado Zona horaria: UTC-3 (Buenos Aires)
